# Kontora24 — Финальный план улучшений v3.0

> Дата: 01.05.2026
> Архитектура: Bitrix24 (CRM) ↔ Kontora24 (MES/WMS + аналитика)
> Принцип: менеджер работает в Bitrix, производство в Kontora24, аналитика в обоих

---

## Архитектура

```
BITRIX24 (CRM)                         KONTORA24 (MES/WMS + Аналитика)
──────────────                         ──────────────────────────────────

Менеджер: сделки, клиенты, КП           Производство: очереди, склад, тех карты
        │                               Аналитика: выручка + производственная
        ▼
Стадия "В производство" ──webhook──►  Автосоздание заказа + авторасчёт
                                              │
                                      design → print → assembly
                                              │
                                      Статус "done" ──webhook──► Bitrix: стадия + цена
                                      
        Каждая смена статуса ──────────────────────► Bitrix timeline
```

## Топ-5 с максимальным эффектом

1. **Bitrix24 интеграция** — убирает двойной ввод, ядро всего workflow
2. **Редактирование заказов** — read-only → полноценное управление
3. **Поиск + пагинация** — без этого нельзя при 100+ заказах
4. **Настройки из БД** — цены на материалы меняются постоянно
5. **Единая производственная доска** — полная картина для менеджера
