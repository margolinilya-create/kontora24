# Kontora24 — План улучшений v1.0

> Автор: SaaS-архитектор · Дата: 01.05.2026
> Текущий статус: ~60% MVP, сильный фундамент, критичные пробелы в permissions, UX и scalability

---

## Принцип приоритизации

Каждое улучшение помечено:
- **P0** — блокер, без этого нельзя пускать пользователей
- **P1** — важно для ежедневной работы
- **P2** — улучшает UX и производительность
- **P3** — nice-to-have, полировка

## 1. LOGIN (`/login`)

### Текущее состояние
Форма email/password → Supabase Auth → redirect на Dashboard.

### Улучшения

| # | Приоритет | Что | Зачем |
|---|-----------|-----|-------|
| 1.1 | P0 | **Показывать "Неверный email или пароль"** вместо сырого Supabase error | Сейчас юзер видит "Invalid login credentials" на английском |
| 1.2 | P1 | **"Запомнить меня"** чекбокс (persistent vs session storage) | Чтобы печатникам/сборщикам не логиниться каждый раз |
| 1.3 | P1 | **Логотип компании** на странице входа вместо буквы "K" | Брендинг |
| 1.4 | P2 | **Сброс пароля** — ссылка "Забыли пароль?" → Supabase resetPasswordForEmail | Базовая функциональность auth |
| 1.5 | P3 | **Loading state** — disabled кнопка + spinner внутри кнопки | Сейчас текст меняется на "Вход...", но нет визуального индикатора |

## Порядок реализации (sprints)

### Sprint 1: "Можно работать" (P0, ~2 дня)
1. Settings — чтение/запись настроек калькулятора
2. Управление пользователями — список, смена ролей
3. Редактирование заказа — notes, deadline, assigned_to
4. Поиск + пагинация на Orders
5. Error boundaries
6. Валидация калькулятора (Zod)

### Sprint 2: "Удобно работать" (P1, ~3 дня)
1. Ролевые дашборды
2. Комментарии к заказам
3. Прикрепление файлов (Supabase Storage)
4. Единая производственная доска (kanban)
5. История движений склада
6. Страница клиента
7. Breadcrumbs + счётчики в навигации
8. Фильтр по периоду в аналитике

### Sprint 3: "Профессионально" (P2, ~3 дня)
1. Kanban-вид заказов
2. Массовые действия
3. PDF экспорт
4. Графики расхода материалов
5. Автосписание материалов
6. Drag-and-drop в очередях
7. PWA

### Sprint 4: "Полировка" (P3, ~2 дня)
1. Тёмная тема
2. Telegram уведомления
3. Экспорт CSV/Excel
4. QR-коды
5. Сохранённые фильтры
6. История расчётов
